[[fashion-mnist]](https://github.com/zalandoresearch/fashion-mnist)
[[Pascal VOC]](http://pascallin.ecs.soton.ac.uk/challenges/VOC/)

* Structure of Datasets for image classification should like the following: 
```
└─my_dataset
    ├─train
    │  ├─animal
    │  ├─flower
    │  ├─guitar
    │  ├─houses
    │  └─plane
    │  └─...
    └─val
        ├─animal
        ├─flower
        ├─guitar
        ├─houses
        └─plane
        └─...
```
